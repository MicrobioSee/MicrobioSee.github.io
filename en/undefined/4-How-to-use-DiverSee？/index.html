4. How to use DiverSee？

About DiverSee

  • DiverSee focuses on the visualization of microbial diversity.

  • It is divided into two parts, α diversity and β diversity.

  • In α diversity, we introduce the pirate diagram into it, which has more elements than the commonly used violin diagram, and can present our data more comprehensively.

  • As we know, the violin plot consists of a combination of box-line and kernel density plots, which are usually used to describe the data distribution and probability density; the pirate plot consists of box-line, kernel density and scatter plots.

  • In the pirate plot, we see the real distribution of the data. There are studies using the ggplot2 package in R to superimpose a scatter plot on top of the violin plot, which as far as I know requires the user to be proficient in programming skills or six months to learn the programming language if there is no programming language learning experience

  • MicrobioSee mainly addresses the fact that visualizing the results of integrating microbiome data is a major challenge for wet-lab researchers, as programming skills are a huge barrier for the vast majority of wet-lab researchers.

  • If you have a very deep mastery of programming languages like R, python and graphics, then you can join us and make MicrobioSee more popular in this world.
